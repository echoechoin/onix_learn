# 键盘中断

- 0x21 键盘中断向量；

| 端口 | 操作类型 | 用途       |
| ---- | -------- | ---------- |
| 0x60 | 读/写    | 数据端口 用于读取键盘输入的数据 |
| 0x64 | 读       | 状态寄存器 |
| 0x64 | 写       | 控制寄存器 |


## 键盘输入的数据原理

按下/松开都会产生中断，而且一直按着不松开也会一直产生中断。
